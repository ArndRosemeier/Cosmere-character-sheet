## Troubleshooting

**"Roller won't open"**
- Make sure you're pressing **D** (not in a text field)
- Try clicking the ğŸ² button instead

**"No skill quick rolls showing"**
- Need a character loaded with skills assigned
- Roller only shows skills with ranks or positive attributes

**"Custom roll not working"**
- Check notation: Must be `XdY` or `XdY+Z` format
- Examples: `1d20`, `2d6+5`, `3d8-2`
- Letters must be lowercase: `d` not `D`

**"History disappeared"**
- History clears when you close the browser
- Click "Clear History" if you want fresh start
- Not saved to character file

**"Plot Die confusing"**
- Think of it as automatic advantage/disadvantage
- 6 = Good luck (advantage)
- 1 = Bad luck (disadvantage)
- 2-5 = Normal
- GM decides when to use it
